# Single-fiber multi-omics and functional profiling ICU-AW

Github repository with all code used for this manuscript (provide link once available). The repository is maintained by Alexandra Winant.

## Description of folder and file contents

The following folders contain:

-   `icu-aw-figures/` : contains all figures included in publication

-   `objects/` : contains raw count matrices and datasets generated during the analysis

-   `scripts/` : contains all scripts used during the analysis. To ensure packages and objects are properly loaded, scripts should be run in the order of rna, protein, integration/QC.

## Resources

Access to raw data, such as RNA-seq fastq and mass spectrometer output files can be found in PRIDE (link) and EGA (link) repositories.
